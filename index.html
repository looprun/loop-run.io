CFP

Loop Run Barcelona conference 2020 is a brand new conference, where everything related to PHP, non-blocking and reactive programming will happen during a whole day.

In order to make the conference as interesting as we want to, we encourage absolutely everyone to submit, at least, one talk about these topics.

The more proposals we have, the more diverse we will be. We encourage you to send us some!

Here you have a small list with some ideas

  • A talk about an existing library / framework
  • A prove of concept you've done as a side project
  • How these programming paradigm can change the language
  • How to refactor an application with some libraries
  • About the programming paradigm.
  • About the performance. Benchmarking included
  • Testing with Promises
  • Promises Vs. Coroutines
  • How could we standardize all this stuff?
